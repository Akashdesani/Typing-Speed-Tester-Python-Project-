# Typing Speed Tester (Python Project)

This project measures typing speed and accuracy.

## Features

- Random sentence generator
- Typing speed calculation
- Accuracy calculation
- Time measurement

## Technologies Used

- Python
- Time Module
- Random Module

## How to Run

1 Install Python

2 Run command:

python typing_speed.py
